# Switch application language

You can switch between these languages:

* <u>English (Default)</u>
* <u>German</u>
* <u>Russian</u>
* <u>Spanish</u>
* <u>Portuguese</u>
* <u>Italian</u>
* <u>Polish</u>
* _French_
* _Ukrainian_
* _Dutch_

<p class="warning">
  Languages, which are italic in this list are not (completly) done yet!
</p>

To switch a language go to "Help > Language > \[the language\]".
