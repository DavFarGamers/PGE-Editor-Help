Music and sound formats support

The PGE supports next audio-formats:

Format Description Notes
WAV Microsoft PCM, Uncompressed audio Support by SMBX
VOC Creative Labs Audio File  
MP3 MPEG-2 Layer 3, Lossy data compressed audio Support by SMBX
OGG OGG Vorbis, Lossy data compressed audio  
FLAC Free Lossless Audio Codec, Loss-less compressed  
MIDI Music Instrument Digital Interface, commands list Support by SMBX
  Modules music formats:  
669 Composer 669, Unis 669  
AMF DSMI Advanced Module Format  
AMF ASYLUM Music Format V1.0  
APUN APlayer  
DSM DSIK internal format  
FAR Farandole Composer  
GDM General DigiMusic  
IT Impulse Tracker  
IMF Imago Orpheus  
MOD 15 and 31 instruments  
MED OctaMED  
MTM MultiTracker Module editor  
OKT Amiga Oktalyzer  
S3M Scream Tracker 3  
STM Scream Tracker  
STX Scream Tracker Music Interface Kit  
ULT UltraTracker  
UNI MikMod  
XM FastTracker 2  
  Game Music Emulators formats:  
AY ZX Spectrum/Amstrad CPC
GBS Nintendo Game Boy  
GYM Sega Genesis / Mega Drive  
HES NEC TurboGraphx-16 / PC Engine  
KSS MSX Home Computer / Other Z80 systems
(not supported FM Sound)		  
NSF Nintendo NES/Famicom (with VRC 6, Namco 106)  
NSFE Nintendo NES/Famicom (with FME-7)  
SAP Atari systems using POKEY sound chip  
SPC Super Nintendo / Super Famicom  
VGM Sega Master System / MarkIII / Sega Genesis / Mega Drive / BBC Micro.  
VGZ Sega Master System / MarkIII / Sega Genesis / Mega Drive / BBC Micro. (Pre-compressed with GnuZIP)  
IMF Id-Software Music File, raw AdLib commands sequence. Required ADLMIDI be toggled!  

 

Note: Same formats are supported by SMBX-38A since 1.4.2 which gets a using of the SDL Mixer X library
